# VPNMvp (WPF) - client


## Run
1. Ensure backend is running: from repo root `backend-mock` run `npm start` (http://localhost:4000)
2. Build & run client


```bash
cd client-windows/VPNMvp
dotnet restore
dotnet build
dotnet run
```
